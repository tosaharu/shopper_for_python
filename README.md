# Shopper  
  
Release: 2023/1/19  
URL: https://hasa.pythonanywhere.com/shopper/  
  
  
## アプリ概要  
  
自分のお得な買い物リストを作れる、家計応援アプリです。  
  
「なるべくおトクに買えるお店を知りたい・・・」  
「商品ごとに行くべきお店の一覧が欲しい・・・」  
  
  
そんな **とにかく安く、効率よく** 買い物をしたいあなたにピッタリ！  
  
## 使い方

1. 自分が買った/気になっている商品の情報を投稿する
1. その商品の品目ごとに、自分の投稿情報と他のユーザーの投稿情報リストが生成される

 - 問い合わせフォーム
